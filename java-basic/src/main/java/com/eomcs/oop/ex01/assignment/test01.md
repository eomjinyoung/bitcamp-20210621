# 과제(2018-03-07)

사용자로부터 입력 받은 성적 데이터(이름, 국어, 영어, 수학)를 
클래스 문법을 이용하여 만든 메모리에 저장하라. 
또한 평균, 합계를 구하고 출력하라. 
입력과 출력에 반복문을 적용하라!

```
> java -classpath bin step03.assignment.Test01
입력: 홍길동 100 100 100

홍길동 100 100 100 300 100.0
임꺽정  90  90  90 270  90.0
유관순  80  80  80 240  80.0
>
``` 
